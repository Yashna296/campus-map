# Campus Map

Campus Map is an AI-powered web application that helps students, visitors, and faculty navigate a college campus easily. It provides an interactive campus map, building search, detailed location information.

## Features

* Interactive campus map
* Building and classroom search
* Building information
* AI chatbot
* User-friendly interface
  
## Tech Stack
* React
* FastAPI
* PostgreSQL
* Python
* Git & GitHub
