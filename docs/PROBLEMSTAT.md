# Problem Statement

Students, visitors, and new faculty members often find it difficult to locate classrooms, laboratories, libraries, cafeterias, and other important places on a college campus. Most campuses only provide static maps or signboards, which are not always easy to understand.

**Campus Map** aims to solve this problem by providing an AI-powered navigation system that helps users find locations quickly through an interactive map and a smart AI assistant. The system will make campus navigation easier, faster, and more convenient for everyone.
